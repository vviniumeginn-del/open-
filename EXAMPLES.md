# wechat-jiaju-writer 使用示例

本文档提供两个完整示例，展示家具厂商品牌内容的创作流程与最佳实践。

---

## 示例1：小户型沙发选购指南（品牌导向）

### 用户输入

```
写一篇小户型沙发选购指南，面向租房与刚需人群，突出耐用、易打理和高性价比
```

### 执行流程

#### 第1步：理解需求
- **主题类型**：家具选购 + 攻略
- **核心角度**：尺寸、材质、结构与维护
- **目标读者**：租房、小户型、新婚刚需
- **品牌角色**：家具厂商，强调品质与服务

#### 第2步：搜索信息（3-4轮）
- 第1轮："小户型 沙发 尺寸 建议"、"沙发 坐深 坐高 标准"
- 第2轮："猫抓布 vs 科技布 vs 真皮"、"沙发 框架 实木 vs 金属"
- 第3轮："沙发 保养 维护"、"租房 沙发 推荐"

#### 第3步：抓取与提炼
- 尺寸建议：2.0-2.4m范围、坐深/坐高区间
- 材质对比：耐磨、透气、清洁难度
- 结构要点：框架、弹簧、填充
- 维护成本：清洁方式与易损点

#### 第4步：改写创作（约2200字）
结构示例：
1. 引子：小户型空间焦虑
2. 需求洞察：租房/刚需的真实痛点
3. 解决方案：尺寸、坐深、模块化布局
4. 产品亮点：材质、结构、耐用性
5. 购买与服务：预算建议、交付周期、售后
6. 总结与行动：预约咨询/到店体验

#### 第5步：封面图（强制）
配色：米白 → 浅木色
提示词要点：空间感、柔光、简洁字体

#### 第6步：内容配图（可选）
若涉及材质对比 → 生成1张材质对比图

#### 第7步：输出
- `小户型沙发选购指南.md`
- `cover.png`
- `material-comparison.png`（可选）

## 示例2：原木风餐桌新品解读（品牌故事）

### 用户输入

```
写一篇原木风餐桌新品解读，突出环保用材、工艺细节与家庭场景
```

### 执行流程

#### 第1步：理解需求
- **主题类型**：新品解读 + 品牌内容
- **核心角度**：材质来源、工艺细节、使用场景
- **目标读者**：新房装修、注重质感人群

#### 第2步：搜索信息（3轮）
- "原木 餐桌 选材 环保"、"FSC 木材 说明"
- "榫卯 工艺 结构"、"餐桌 尺寸 建议"
- "原木 风格 搭配"、"餐厅 氛围 软装"

#### 第3步：抓取与提炼
- 选材：环保认证、木种特点
- 工艺：结构稳定性、耐用性
- 体验：触感、观感、耐污性
- 场景：家庭聚餐、日常使用

#### 第4步：改写创作（约2000字）
结构示例：
1. 引子：餐桌是家的“中心”
2. 需求洞察：家庭场景与耐用诉求
3. 解决方案：木种选择与结构细节
4. 产品亮点：工艺、质感、使用体验
5. 购买与服务：交付、保养、售后
6. 总结与行动：到店体验/咨询

#### 第5步：封面图（强制）
配色：奶油白 → 雾金
视觉元素：木纹纹理、柔光

#### 第6步：内容配图（可选）
若涉及结构/材质对比 → 生成1张对比图

#### 第7步：输出
- `原木风餐桌新品解读.md`
- `cover.png`

---

**提示**：示例中的品牌信息用占位符替换，正式输出时需填入品牌名、产品线、卖点与服务信息。
