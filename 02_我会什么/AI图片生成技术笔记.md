## 镜头内容、光线、风格、环境和细节

**（右上角标注：风格匹配）**

### 核心要素

- **景别角度、焦距、背景**：景别是远、中、近、特、微特。焦距越长背景越虚幻、景深越低、清晰。

- **主体动作、情绪**：图中的主体做什么动作，有什么情绪。

- **光源位置、光线方向、光线质感、氛围时间**

- **构图、色彩、整体风格**

- **场景、材质、细节纹理**

## 系列提示词、建立一个整体的视觉系统、镜头十八风格

### 流程与方法

- **流程塑造画面**：把握用固定风格的，可以固定图片风格的系列图片。

- **图片倒推进行反推提示词**：经常出现不准确，就是整体风格的不准确，没有给定风格。

- **传出一个准确的风格**：比如大师风格立刻靠近，更多是在于风格上的调整，能够与匹配风格找到加进去。

- **然后选定好的生图模型生图**

## Json prompt 指令

### Nano Banana 的优势

**Json prompt 指令下 Nano Banana 这种指令跟随优秀就可以把提示词写的很详细**

- **Json prompt 结构化方便修改**：结构可以自然语言对控制。

- **自然语言代表 MJ（Midjourney）出图质量最好**：这种自然语言模型接到提示词的详细反映是紧密结合的。

- **最好方式**：利用 MJ 出图，再利用 Nano Banana 进行瑕疵修改。

- **精准和细微控制**：依旧还是线稿进行限制和蒙版，只有这样才能准确率高。
