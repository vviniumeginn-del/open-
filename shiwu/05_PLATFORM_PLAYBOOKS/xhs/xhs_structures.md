# 小红书正文结构模板

## 1. 单品审美种草型

结构：
1. 开头点名"被问爆/反复询问"
2. 审美叙事（风格+质感+文化意象）
3. 情绪价值（团聚/氛围）
4. 总结+标签

示例：圆餐桌文案

---

## 2. 全屋搭配清单型

结构：
1. 总引子（性价比/整套方案）
2. 分空间分品类拆解（卧室/客厅/餐厅）
3. 每项2-4条卖点
4. 标签收尾

示例：全屋白蜡木家具文案

---

## 3. 合集盘点型

结构：
1. 人群/风格划分
2. 推荐材料/产品
3. 推荐款式/店铺
4. 标签收尾

示例：30款中古床、品牌大盘点

---

## 4. 收纳卖点密集型

结构：
1. 多款产品并列
2. 每款"核心场景 + 容量/结构 + 小户型友好"
3. 高频可视化用词（分区、抽屉、隔板、占地）
4. 标签

示例：中古收纳柜合集

---

## 5. 工厂背书型

结构：
1. 悬念/被问爆开头
2. 工厂实拍背书
3. 类目展示（床具/柜体/小物）
4. 预算优势/下期预告/互动CTA
