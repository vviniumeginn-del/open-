# 资料池 (intake)

所有外部搜集、Agent整理的内容，先入这里，不直接入主库。

## 子目录

| 目录 | 用途 |
|------|------|
| 01_raw_inputs | 手工输入的原始资料 |
| 02_agent_collected | Agent 搜集的资料 |
| 03_structured_drafts | Agent 整理后的结构化草稿 |
| 04_review_queue | 等待审核/待确认 |
| 05_rejected_or_pending | 暂不采纳/低质量 |

## 流程

1. 资料先入 `01_raw_inputs` 或 `02_agent_collected`
2. Agent 整理后移至 `03_structured_drafts`
3. 需人工确认的移至 `04_review_queue`
4. 审核通过后移入主库对应目录
5. 不采纳的移至 `05_rejected_or_pending`
